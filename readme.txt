Project : Uni Equipment


1-Create Anaconda Environment
 > Install Anconda from Browser if not installed in your system
 > Open VS code and open terminal as Command Prompt
 > conda create -- myenv python=3.9
 > activate myenv

2-Installation
 > open terminal as PowerShell in VS code
 > pip install -r requirements.txt
 > python manage.py makemigrations
 > python manage.py migrate

3-Usage
 > in same terminal start server -> python manage.py runserver
